1. LOGIN AS CUSTOMER
   → Dashboard → Click "My Vehicles" in sidebar
   → Empty state → Click "Add Your First Vehicle"
   → Fill form → KA01AB1234, 4-Wheeler, Honda City, 2022
   → Vehicle card appears with colored header
   → Click View → Vehicle detail page with policy info
   → Click Edit → Update model name
   → Click Delete → Confirmation modal → Delete

2. LOGIN AS ADMIN
   → Sidebar → Vehicles
   → See ALL vehicles from ALL customers
   → Stats cards (Total, 2-Wheeler, 4-Wheeler, Commercial)
   → Search, filter by type, sort, pagination
   → Click "Add Vehicle" → Need to provide Customer ID
   → View/Edit/Delete any vehicle

3. LOGIN AS STAFF
   → Same as Admin but via /staff/vehicles
   → Can add vehicles for customers
   → Can view and edit vehicles
